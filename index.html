Como é uma estrutura básica de um HTML?

A estrutura de um documento HTML é composta por três partes principais:

A estrutura principal, o cabeçalho e o corpo:

A estrutura principal:
Começa com a etiqueta < HTML > e termina com a etiqueta < /HTML > e envolve todas as outras.

O cabeçalho:
Contém informações sobre o documento, como o título, e são utilizadas as etiquetas < HEAD > e < /HEAD >.

O corpo:
Contém o conteúdo da página, como texto, imagens, sons, formulários, etc., e fica entre as etiquetas < BODY > e < /BODY >.

  • Índice

    • Entendendo a diferença entre listas ordenadas e não ordenadas

    Lista ordenada: Enumera os elementos da lista, atribuindo um número ou letra a cada um, é usada quando a sequência dos itens é importante, para uma lista ordenada é utilizada a tag < ol>< /ol> seguida da tag < li>< /li>.

    Lista não ordenada: Coloca marcadores antes de cada elemento, sem utilizar numeração. É ideal para listas onde a ordem não importa, como listas de tarefas ou tópicos, é utilizada as tag < ul>< /ul> seguidas da tag < li>< /li>.

  • Índice

    • Temos também os elementos de cabeçalho que vai do H1 ao H6 devendo ser utilizado de acordo com a importância do título.
    Podemos usar a tag < strong>< /strong> em torno de uma palavra ou frase quando quisermos tornala negrito, e a tag < i>< /i> para tornar o texto itálico, também podemos utilizar a tag < u>< /u> para sublinhar um texto.
